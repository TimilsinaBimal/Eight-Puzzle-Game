## Eight Puzzle Game

Solution of Eight Puzzle game using Breadth First Search(BFS).   
Make sure Graphviz is installed in your system to run this project.

#### Installation Methods

**Using pip**   
```pip install graphviz```

**Using Chocolatey**    
```choco install graphviz```


#### Run
```$ python solution.py```
